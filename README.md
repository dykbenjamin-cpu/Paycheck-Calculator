# Paystub Calculator

Simple static web app to estimate a paystub amount.

## Inputs
- Hourly wage
- Hours worked
- Vacation days (paid)
- Sick days (paid)
- Pension:
  - Percent of gross pay, or
  - Fixed dollars per hour worked
